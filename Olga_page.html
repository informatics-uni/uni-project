First of all we made a query on the information present on the topic of architecture in Wikidata.
RESULT 		SELECT DISTINCT ?item ?itemLabel ?locationLabel WHERE { ?item wdt:P31/wdt:P279* wd:Q839954; # instance of (or subclass) archaeological site wdt:P17 wd:Q38; # country = Italy wdt:P131 ?location. # located in administrative unit SERVICE wikibase:label { bd:serviceParam wikibase:language "it,en". } } LIMIT 80
A second query retrieves castles and cathedrals located in Italy, optionally showing their Commons categories and types. SELECT DISTINCT ?item ?itemLabel ?typeLabel ?commonsCategory WHERE { { ?item wdt:P31 wd:Q23413. # castles } UNION { ?item wdt:P31 wd:Q2977. # cathedrals } ?item wdt:P17 wd:Q38. # country = Italy OPTIONAL { ?item wdt:P373 ?commonsCategory. } # Wikimedia Commons category OPTIONAL { ?item wdt:P31 ?type. } SERVICE wikibase:label { bd:serviceParam wikibase:language "en,it". } } LIMIT 50
